# Filament Admin Theme Redesign

## Current Setup

**Current Theme:**
- Primary Color: `Color::Amber` (default Filament amber)
- Brand Name: "RadarLeb Admin"
- Brand Logo: `assets/imgs/logo.png`
- View: Default Filament login view

**RadarLeb Brand Colors (from app.css):**
- Primary Blue/Cyan: `#6bbace` (rgb(107, 186, 206)) / `#66afdb` (rgb(102, 175, 219))
- Accent Red: `#ff7676` (rgb(255, 118, 118)) / `#ea3323` (rgb(234, 51, 35))
- Success Green: `#06ac06` (rgb(6, 172, 6))
- Dark Background: `#000000` / `rgba(0, 0, 0, 0.5)`
- Gradient: `linear-gradient(to top, #6bbace, #ff7676)`

## Theme Options

### Option 1: Light Theme (Clean & Modern)

**Color Palette:**
- Primary: Blue/Cyan (`#66afdb` / rgb(102, 175, 219))
- Background: Light gray/white
- Accents: Subtle blue tones
- Text: Dark gray on light

**Characteristics:**
- Clean, professional appearance
- High contrast for readability
- Modern flat design
- Suitable for daytime use

### Option 2: Dark Theme (Brand Accent)

**Color Palette:**
- Primary: Blue/Cyan (`#6bbace` / rgb(107, 186, 206))
- Background: Dark gray (not pure black)
- Accents: Red highlights (`#ff7676`)
- Text: Light on dark

**Characteristics:**
- Matches RadarLeb game aesthetic
- Modern dark UI
- Brand-consistent colors
- Eye-friendly for extended use

## Recommended: Option 2 (Dark Theme)

Matches the RadarLeb brand better and aligns with the game's dark aesthetic.
